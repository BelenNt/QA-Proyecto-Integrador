Este repositorio simula el trabajo de una QA (Francis Belen) dentro de un proyecto real, 
aplicando buenas prácticas básicas de testing, organización y control de calidad de software.

Este repositorio incluye:
- **Casos de prueba** documentados para validar funcionalidades.
- **Evidencias de testing**, que respaldan la ejecución de los casos de prueba.
- **Documentación básica** que explica la estructura y el propósito del proyecto.

Esto sirve para practicar y demostrar el uso correcto del flujo de trabajo de QA, 
aplicando criterio, orden y documentación clara en un entorno similar al de un proyecto real.
